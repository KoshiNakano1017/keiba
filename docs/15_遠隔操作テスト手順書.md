# 遠隔操作テスト手順書

## 1. はじめに
このドキュメントでは、Windows PCにおけるリモートアクセスの設定およびテスト手順を説明します。使用するツールは次の通りです。
- TeamViewer
- AnyDesk
- Chrome Remote Desktop

## 2. ツールの準備
### 2.1 TeamViewer
1. TeamViewerを公式サイトからダウンロードし、インストールします。
2. 起動後、パートナーIDとパスワードを確認します。

### 2.2 AnyDesk
1. AnyDeskを公式サイトからダウンロードし、インストールします。
2. 起動後、アドレス番号を確認します。

### 2.3 Chrome Remote Desktop
1. Chromeブラウザを起動し、ChromeウェブストアでChrome Remote Desktopを追加します。
2. リモートアクセスを有効にし、設定を行います。

## 3. 接続手順
### 3.1 TeamViewerでの接続
1. 相手方のパートナーIDを入力します。
2. 接続を開始し、パスワードを入力します。

### 3.2 AnyDeskでの接続
1. 相手方のアドレス番号を入力します。
2. 接続リクエストを送信し、相手方で承認を待ちます。

### 3.3 Chrome Remote Desktopでの接続
1. あなたのPCからChrome Remote Desktopを開き、接続したいPCを選択します。
2. PINを入力して接続します。

## 4. Pythonによるバッチ実行
リモート接続後、Pythonスクリプトをバッチファイルで実行できます。

1. スクリプトを準備します (例: `script.py`) 。
2. コマンドプロンプトを開き、以下のコマンドを入力します。
   ```bash
   python script.py
   ```

## 5. Supabaseでの検証
1. Supabaseの公式サイトにアクセスします。
2. アカウントにログインし、必要なデータベーステーブルを確認します。
3. スクリプト実行後のデータを検証します。

## 6. まとめ
この手順書に従って、Windows PCにおけるリモートアクセスが簡単に設定できます。各ツールの使い方を理解し、円滑なテスト環境を実現しましょう。
